# Cyber Security Projects
==> Incident Analysis and Response Project

This project implements an Incident Analysis and Response environment using SOC (Security Operations Center) principles.  
The main goal is to understand the full incident response lifecycle through a simulated real-world environment.

🛠️ Environment Setup
We used two virtual machines:
- Ubuntu (Wazuh server)
- CentOS (Wazuh agent)

🔧 Steps Performed
1. Installed and configured Wazuh on Ubuntu.
2. Installed Wazuh agent on CentOS and linked it to the manager.
3. Simulated Brute Force and DDoS attacks.
4. Monitored and analyzed logs using Wazuh and Wireshark.
5. Attempted to implement active response (though faced some issues).
